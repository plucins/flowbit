Text block

Text blocks should carry the main narrative, with generous line height and warm contrast against the canvas.

Lists remain simple and semantic, while styles stay tied to global tokens.

  • Use one visual rhythm across all paragraphs.
  • Keep spacing and emphasis token-based.
  • Avoid local one-off colors in content blocks.
